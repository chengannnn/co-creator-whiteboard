## Codebase Patterns
- Closed shapes (rectangle, ellipse, rhombus) share `createBBoxHandler` factory in RectangleHandler.ts — all use `Math.min`/`Math.abs` for coordinate normalization
- Vector shapes (line, arrow) preserve direction via `points` array with signed deltas; `width`/`height` store absolute values separately
- Freehand normalizes by recomputing bbox from points and shifting all points relative to min values
- Element creation uses `createBaseElement` helper in helpers.ts which initializes `groupIds: []`
- Three-layer canvas: bg (background), static (committed shapes), interactive (draft/in-progress shapes)
- Resize uses absolute coordinate calculation: `applyResize` receives `mouseX`/`mouseY` and recalculates dimensions from anchor corner (CanvasComponent.tsx + SelectToolHandler.ts)
